This is the complete list of slash commands. For an overview by category, see [Slash Commands](/docs/concepts/commands).

Commands marked **Hosted** require [managed hosting](/docs/guides/hosting) (knowledge graph).

## Core

| Command | Description |
|---------|-------------|
| `/activity` | See what's happening across the team |
| `/dashboard` | Your personal status and recent work |
| `/handoff` | Leave structured notes for others (auto-saves) |
| `/save` | Commit, push, and PR ‚Äî across all repos with changes |
| `/todo` | Manage personal tasks |
| `/wrap` | Close your session with a personal summary |

## Knowledge

| Command | Description |
|---------|-------------|
| `/add` | Add a document, link, or artifact to shared memory |
| `/archive` | Store effective prompt patterns as reusable knowledge |
| `/deep-reflect` | Cross-reference an insight against the knowledge base |
| `/meeting` | Analyze meetings from Granola |
| `/note` | Save a personal, half-baked thought |
| `/reflect` | Capture a share-ready decision, pattern, or insight |

## Coordination

| Command | Description |
|---------|-------------|
| `/announce` | Draft and send a message to the Telegram group |
| `/ask` | Route a question to self, a person, or the team |
| `/delete-user` | Remove a member from this Egregore |
| `/invite` | Add someone (GitHub invitation + setup link) |
| `/issue` | Report something broken |
| `/quest` | Start or contribute to an exploration |
| `/quest-suggest` | Analyze quest drift and suggest restructuring |

## Identity

| Command | Description |
